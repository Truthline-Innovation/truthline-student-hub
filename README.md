# TruthLine Student Hub

A mini startup project built by members of TruthLine Innovations.

## Purpose
To help students showcase projects, learn collaboration, and practice
full-stack software development.

## MVP Features
- User registration & login
- Project upload (title, description)
- Project listing page
- Basic user dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Contribution Flow
- Each feature is developed on a branch
- Pull Requests are reviewed before merge
- Discussions used for planning
